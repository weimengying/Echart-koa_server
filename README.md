# Echart-koa_server
Vue+ECharts来展示电商平台数据
#### 前端技术栈采用Vue+VueRouter+Axios+Echarts
#### 后端技术栈采用Node.js+koa组成
### 前端
前端项目：vision
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 后端服务：koa_server
npm install
node app.js
